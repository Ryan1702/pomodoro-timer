# 🍅 番茄钟

一个简约、优雅的番茄钟桌面应用，基于 Electron 构建，助你高效专注。

## ✨ 功能

- **三阶段循环** — 专注 → 短休息 → 长休息，自动流转，长休息间隔可配置
- **环形进度条** — SVG 圆环实时显示剩余时间，支持纯色与渐变双模式
- **12 款主题** — 经典番茄 / 海风青 / 樱花粉 / 落日金 / 薰衣草紫 / 石墨灰 / 春日嫩芽 / 薄荷冰泉 / 极光幻境 / 桃气满满 / 午夜星辰 / 落日熔岩，专注用金币解锁心仪配色
- **金币系统** — 每完成 1 分钟专注赚 1 🪙，攒够金币解锁付费主题（经典番茄免费）
- **永久统计** — 总专注分钟数和金币数持久化存储，不受周期重置影响
- **紧凑模式** — 一键缩小至 200×160 迷你窗口，不占屏幕空间
- **窗口置顶** — 一键将番茄钟固定在最上层，始终可见不被打断
- **可调时长** — 点击齿轮图标，自由设置专注、短休息、长休息时长（1–120 分钟）
- **每日跳过额度** — 专注阶段每天可跳过 3 次，凌晨 0 点自动刷新；休息阶段无限制
- **系统通知** — 阶段结束时弹出原生通知，不错过节奏
- **数据持久化** — 时长设置、主题偏好、金币与分钟统计、解锁主题列表、每日跳过配额自动保存，下次打开无需重新配置

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

### 打包发布

```bash
# macOS
npm run build:mac

# Windows
npm run build:win
```

构建产物输出到 `dist/` 目录。

## 📁 项目结构

```
main.js              # Electron 主进程：窗口管理、IPC、通知、金币信号文件
preload.js           # 上下文桥接，安全暴露 API 给渲染进程
renderer/
  index.html         # 界面结构（纯 HTML，无框架）
  style.css          # 样式 + CSS 变量主题系统（12 款主题）
  app.js             # 核心番茄钟逻辑（状态机、计时、阶段切换、金币、主题商城）
assets/
  logo.png           # 应用图标（构建用）
package.json         # 项目配置 + electron-builder 打包配置
```

## 🛠 技术栈

- **Electron** — 跨平台桌面框架
- **原生 DOM** — 无前端框架依赖，轻量高效
- **SVG** — 环形进度条动画（支持 linearGradient 渐变）
- **localStorage** — 本地设置 + 统计 + 主题数据持久化
