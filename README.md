# 🍅 番茄钟

一个简约、优雅的番茄钟桌面应用，基于 Electron 构建，助你高效专注。

## ✨ 特性

- **三阶段循环** — 专注 → 短休息 → 长休息，自动流转
- **环形进度条** — SVG 圆环实时显示剩余时间，三色主题随阶段切换（红/绿/蓝）
- **可调时长** — 点击齿轮图标，自由设置专注、短休息、长休息时长（1-120 分钟）
- **窗口置顶** — 一键将番茄钟固定在最上层，始终可见不被打断
- **系统通知** — 阶段结束时弹出原生通知，不错过节奏
- **番茄计数** — 记录当前周期完成的番茄数，长休息间隔可配置
- **数据持久化** — 时长设置自动保存，下次打开无需重新配置

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

或双击项目根目录下的 `番茄钟.command` 直接启动。

## 📁 项目结构

```
main.js              # Electron 主进程：窗口管理、IPC、通知
preload.js           # 上下文桥接，安全暴露 API 给渲染进程
renderer/
  index.html         # 界面结构（纯 HTML，无框架）
  style.css          # 样式 + CSS 变量主题系统
  app.js             # 核心番茄钟逻辑（状态机、计时、阶段切换）
assets/
  icon.svg           # 应用图标
```

## 🛠 技术栈

- **Electron** — 跨平台桌面框架
- **原生 DOM** — 无前端框架依赖，轻量高效
- **SVG** — 环形进度条动画
- **localStorage** — 本地设置持久化
